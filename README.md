# lidebot

:robot: que compara as manchetes dos principais jornais do Brasil de hora em hora.

O projeto, atualmente, coleta manchetes de **6 jornais** e essa lista deve expandir em breve.

- [x] Folha de S. Paulo
- [x] O Globo
- [x] UOL
- [x] Valor Econômico
- [x] Estado de S. Paulo
